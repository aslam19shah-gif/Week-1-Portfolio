# Personal Portfolio Website

## Overview

This project is a responsive three-page personal portfolio website built using HTML, CSS, and JavaScript. It was developed as part of foundational web development practice, focusing on semantic HTML structure, clean CSS architecture, responsive layout design, and basic interactivity.

The website includes a Home page, About page, and Contact page with form validation, along with additional sections such as Projects and FAQ.

---

## Features

- Responsive layout (mobile-first approach)
- Semantic HTML structure
- Clean and reusable CSS
- Flexbox-based layout system
- Projects section with card layout
- FAQ section using native HTML accordion (`details` and `summary`)
- Contact form with HTML validation attributes
- Dark mode toggle with localStorage support
- Sticky footer layout using Flexbox
- Deployed using GitHub Pages

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (DOM manipulation, localStorage)
- GitHub Pages (Deployment)

## Running the Project Locally

1. Clone the repository:


git clone https://github.com/your-username/portfolio.git


2. Navigate to the project directory.

3. Open `index.html` in your browser.

No additional dependencies or installations are required.

---

## Deployment

The project is deployed using GitHub Pages.

To deploy:
1. Push the project to a public GitHub repository.
2. Go to Repository Settings → Pages.
3. Select "Deploy from a branch".
4. Choose the `main` branch and `/ (root)` folder.
